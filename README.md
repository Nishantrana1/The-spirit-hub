## 🌟 About The Spirit Hub

**The Spirit Hub** was born from a simple belief: *the best technology is built by people who deeply care — about craft, about each other, and about what they're creating for the world.*

We're a gathering place for the restless minds who debug at midnight, who read RFCs for fun, who open-source their best work because knowledge should be free. Not just a community — **a movement.**

### Our Pillars

| Pillar | Description |
|--------|-------------|
| ◈ **Deep Learning** | Dive deep into topics that matter. No surface-level content here. |
| ◉ **Open Collaboration** | Build together, share freely, and grow as a collective. |
| ◌ **Radical Inclusion** | Everyone belongs. From self-taught developers to seasoned engineers, every perspective matters. |

## 🚀 Getting Started

1. **Clone the repository**
   ```bash
   git clone https://github.com/Nishantrana1/The-spirit-hub.git
   cd The-spirit-hub
   ```

2. **Run locally** (using any static server)
   ```bash
   npx -y http-server . -p 8080 --cors -o
   ```

3. **Open in browser**
   Navigate to `http://localhost:8080`
